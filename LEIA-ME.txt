Projeto React e Node.js com Prisma - Leia-me

Este repositório contém um projeto full-stack desenvolvido com React no frontend, Node.js no backend e Prisma como ORM para interação com o banco de dados. Abaixo, você encontrará informações sobre a estrutura do projeto e instruções para instalação e execução.

.
├── client/                 # Pasta do Frontend (React)
├── server/                 # Pasta do Backend (Node.js)
├── npm.bat                 # Script para instalar as dependências
├── start.bat               # Script para iniciar os servidores do frontend e backend
├── prisma.bat              # Script para iniciar o servidor do banco de dados do Prisma
└── .git/                   # Pasta do sistema de controle de versão Git
